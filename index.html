积分计算规则

+3 分
精准预测： 预测比分与官方完场比分一致（例如预测2-1，实际赛果2-1）。
+2 分
净胜球与胜负预测： 预测对胜负且净胜球一致，但比分不同（例如预测2-0，实际赛果3-1）。
+1 分
胜负/ outcome预测： 仅预测对胜负方向或平局，但净胜球不同（例如预测3-1，实际赛果1-0）。
+1 分
晋级球队预测： 准确预测出最终晋级的球队（含点球大战胜出者）。对平局预测至关重要！

* 注：淘汰赛比分竞猜以加时赛结束（120分钟）时的比分为准。如进入点球大战，完场比分按平局（如1-1）计算，“晋级球队”预测选项将决定点球大战胜出者的积分奖励。

数据库配置 (Supabase)

配置您自己的 Supabase 数据库，与好友实时共享预测数据及排行榜。留空则自动使用本地缓存和链接分享模式。

Table 创建指引 (SQL Editor):

请在您的 Supabase 控制面板中打开 SQL Editor，复制并执行以下命令创建数据表： 

create table predictions (
  username text primary key,
  tips jsonb not null,
  updated_at timestamp with time zone default now() not null
);

alter table predictions disable row level security;
